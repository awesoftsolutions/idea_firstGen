# LanternBird

*A tiny Python toolkit for generating cozy, story-rich scene prompts, mood palettes, and thumbnail compositions from a single visual theme.*

LanternBird is a fictional Python repository for creators who want to turn a vibe into usable creative assets. It is built around a simple idea:

> give the app a character mood and an environment mood, and it produces structured scene concepts you can reuse for games, illustrations, wallpapers, mockups, or writing prompts.

The project leans into two distinct energies:
- playful, expressive, neon-urban character styling
- warm, dreamy, lantern-lit countryside environments

That contrast is the whole point.

---

## Features

- Generate scene briefs from a small set of mood inputs
- Create reusable color palettes for “urban pop” and “cozy dusk” aesthetics
- Build prompt packs for image generation workflows
- Score vibe consistency between a character concept and an environment concept
- Export results as Markdown, JSON, or plain text
- Optional CLI for quick ideation sessions

---

## Example Use Cases

LanternBird is useful for:

- concept artists blocking out thumbnail ideas
- indie game devs exploring tone before production
- writers who want better setting prompts
- UI designers building moodboards
- hobbyists generating wallpaper ideas or stylized scene descriptions

Example themes it handles well:

- neon street portraits
- whimsical mountain villages
- lantern-lit paths
- soft-focus sunset scenes
- cute-but-edgy character mood pairings
- bird, flower, and cobblestone visual motifs
